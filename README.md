# `Angular Reedsy`


- Please download the repo.

- Install dependencies: 
```
npm install 
```

- Run Demo Page: 
```
npm start
``` 
App should be running on  `http://localhost:8000/`

- Documentation: 

If Demo page is running: 
`http://localhost:8000/docs/`

- Run tests: 
```
npm test-single-run
```
